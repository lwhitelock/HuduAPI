# Start-HuduS3Export

## SYNOPSIS
Initiate an S3 Backup of Hudu Data (credentials must be configured in settings)

## SYNTAX

```
Start-HuduS3Export
```

## DESCRIPTION
Kicks off a backup in S3 if you have configured Hudu to do this (if so, backs up every Sunday)
