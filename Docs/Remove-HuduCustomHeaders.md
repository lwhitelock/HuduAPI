# Remove-HuduCustomHeaders

## SYNOPSIS
Remove Custom Headers that are injected into each request

## SYNTAX

```
Remove-HuduCustomHeaders [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Unsets the Hudu Custom Header variable

## EXAMPLES

### EXAMPLE 1
```
Remove-HuduCustomHeaders
```
